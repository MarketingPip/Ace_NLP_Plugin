Ace Editor Plugin Demo with NLP Highlighting

This demo showcases Ace Editor integrated with an NLP (Named Entity Recognition) plugin for highlighting text entities like people, places, dates, etc.

Ace Editor with NLP Highlighting Demo

View Source 

  // Ace Editor Plugin Demo with NLP Highlighting
  import ace from 'https://esm.sh/ace-builds/src-noconflict/ace';
  import nlp from 'https://esm.sh/compromise';
  import * as b from 'https://esm.sh/gh/MarketingPip/Ace_NLP_Plugin/';

  // Initialize Ace Editor
  const editor = ace.edit('editor');

  // Set up the editor theme and language mode
  editor.setTheme('ace/theme/monokai');
  editor.session.setMode('ace/mode/javascript');

  // NLP Patterns for highlighting
  const patterns = {
    '#Date+': 'date',
    '#Person+': 'person',
    '#Place+': 'place',
    '#Organization+': 'organization',
    '#Url+': 'url',
    '#PhoneNumber+': 'phone',
    '#AtMention+': 'atMention',
    '#Email+': 'email',
    '#Currency+': 'currency',
    '#Value+': 'value',
  };

  // Function to highlight text using NLP patterns
  const highlightWords = ace.require('ace/plugin/yaml-validation');

  // Monitor changes in the editor
  editor.getSession().on('change', function () {
    highlightWords(editor, patterns, nlp);
  });

  // Apply NLP highlighting initially
  highlightWords(editor, patterns, nlp);
